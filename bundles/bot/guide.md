Tracegate Guide (пошагово)

Ниже самый простой порядок, чтобы все заработало с нуля.

Шаг 1. Создай устройство
1. Нажми `/start`.
2. Открой раздел устройств.
3. Нажми `Добавить устройство`.
4. Введи понятное имя, например: `iPhone`, `MacBook`, `Home-PC`.

Шаг 2. Создай подключение
1. Выбери только что созданное устройство.
2. Нажми `Добавить подключение`.
3. Выбери нужный тип: `VLESS`, `Hysteria2` или `WireGuard`.
4. Дождись сообщения `Создано: ...`.

Шаг 3. Получи конфиг
1. Открой список подключений устройства.
2. Выбери нужное подключение.
3. Нажми `Выдать конфиг` (или аналогичную кнопку выдачи).
4. Для `VLESS/Hysteria2` ссылка-конфиг приходит отдельным сообщением. Скопируй ее целиком.

Шаг 4. Импортируй в клиент
1. Открой приложение-клиент VPN на устройстве.
2. Выбери `Import from clipboard` или `Import from QR`.
3. Вставь ссылку из бота или отсканируй QR.
4. Включи подключение.

Шаг 5. Проверка
1. После запуска убедись, что интернет работает через туннель.
2. Если не работает: удали профиль в клиенте и снова импортируй из бота.

Полезные команды
- `/start` - главное меню
- `/guide` - показать этот гайд
- `/clean` - попытка удалить последние сообщения бота в чате
- `/cancel` - отменить текущий ввод (если бот ждет текст)

Если застрял
1. Начни заново: `/start`.
2. Проверь, что выбрано правильное устройство.
3. Сгенерируй конфиг еще раз и импортируй по новой.
