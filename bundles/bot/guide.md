Tracegate Guide

Этот текст выводится по команде /guide.

Как редактировать:
- По умолчанию бот читает файл: bundles/bot/guide.md (внутри контейнера: /app/bundles/bot/guide.md)
- Можно переопределить путь через переменную окружения BOT_GUIDE_PATH

Команды:
- /start: главное меню
- /guide: показать гайд
- /clean: очистить последние сообщения в чате (best-effort, зависит от ограничений Telegram)
