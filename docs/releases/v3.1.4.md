# Tracegate 3.1.4

Tracegate 3.1.4 restores the account-bound Telegram Proxy data path through
the Entry host while keeping Endpoint as the outbound egress.

- issue and revoke MTProto grants against the Entry agent in
  `entry-local-endpoint-egress` mode;
- publish the Entry hostname in Telegram Proxy links;
- authorize the dedicated Entry-to-Endpoint VLESS/WebSocket backhaul client
  in the Endpoint Xray bundle;
- keep the Endpoint-local Telemt runtime out of the active production
  manifest for this route mode.
