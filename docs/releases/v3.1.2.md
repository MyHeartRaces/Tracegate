# Tracegate 3.1.2

Tracegate 3.1.2 is the production promotion release for the host-based
deployment contract introduced in 3.1.1. It pins the bundled Xray binary by
version and SHA-256 so branch and tag image builds cannot silently consume a
different upstream release.

Use the immutable GHCR digest recorded in the encrypted private deployment
overlay, not a mutable image tag.
