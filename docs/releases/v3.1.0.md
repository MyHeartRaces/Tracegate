# Tracegate 3.1.0

Tracegate 3.1 consolidates the production hardening completed after the 3.0.0
release while preserving the Entry and Endpoint architecture.

## Highlights

- hardened Entry-to-Endpoint Chain routing and trusted Reality SNI handling;
- resilient Telemt and MTProto Entry ingress with Endpoint egress;
- Hysteria2 Gecko and Salamander stability improvements, including longer UDP
  session retention;
- Endpoint ingress host/SNI allocation and direct-routing corrections;
- improved backup, pressure monitoring, readiness checks and release gates;
- fixes for Shadowsocks, WebSocket, gRPC and client export compatibility.

## Release artifacts

- Python wheel and source distribution;
- generic host runtime bundle archive;
- SHA-256 checksum manifest;
- GHCR image tags `3.1.0`, `sha-<commit>` and `latest`.

The release contains no production values, live addresses, credentials,
certificates, client exports or rendered private runtime configuration.
