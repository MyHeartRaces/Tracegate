# Tracegate 3.1.1

Tracegate 3.1.1 finalizes the host-based production deployment contract. It
replaces the retired k3s/Helm package with immutable Compose profiles for the
control and gateway roles, database-backed readiness, migration and backup
gates, image rollback, private profile preflight and containerized WireGuard
peer synchronization.

The release contains a generic public host-runtime archive. Live inventory,
credentials, domains, addresses and rendered profiles remain exclusively in
the encrypted private deployment repository.
