# Tracegate 3.0.0

Tracegate 3 promotes the Entry and Endpoint architecture as the supported
production model. Endpoint owns stable egress and sensitive runtimes; Entry is
a replaceable ingress surface backed by authenticated, fail-closed links.

## Highlights

- one k3s/Helm deployment contract for control plane, gateways and private
  runtime reloads;
- VLESS Reality, Hysteria2 with Salamander, VLESS gRPC/WebSocket backup paths,
  Shadowsocks-2022 with ShadowTLS v3 and WireGuard-over-WebSocket exports;
- shared Entry TCP/443 MTProto ingress with Endpoint-local official MTProxy;
- Grafana and Prometheus dashboards with bot-issued Grafana access;
- restored bot onboarding, help, administration and connection lifecycle
  flows;
- strict public/private repository boundaries and an automated release privacy
  gate.

## Release Artifacts

- Python wheel and source distribution;
- Helm chart package `tracegate-3.0.0.tgz`;
- generic runtime bundle archive;
- SHA-256 checksum manifest;
- GHCR image tags `3.0.0`, `sha-<commit>` and `latest`.

The release contains no production values, live domains or addresses, TLS
keys, client exports, decoy content, bot credentials or rendered runtime
configuration.
